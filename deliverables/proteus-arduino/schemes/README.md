# Schemes For Proteus 8.13

В этой папке лежат схемы и пошаговые инструкции для сборки лабораторных работ в Proteus 8.13.

## Что внутри

- `lab1_variant1.md` - ЛР1, вариант 1
- `lab1_variant2.md` - ЛР1, вариант 2
- `lab2_variant1.md` - ЛР2, вариант 1
- `lab2_variant2.md` - ЛР2, вариант 2
- `lab3_crc8_reset.md` - ЛР3, версия со сбросом CRC
- `lab3_crc8_accumulate.md` - ЛР3, версия без сброса CRC
- `lab4_variant2.md` - ЛР4, вариант 2

## Как использовать

1. Откройте нужный файл `.md` на GitHub или локально.
2. Добавьте в Proteus перечисленные компоненты.
3. Соедините их по таблице и схеме.
4. Для Arduino Uno укажите `HEX` из папки `../proteus/<variant>/`.
5. Запустите симуляцию.
