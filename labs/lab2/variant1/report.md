# Отчет по лабораторной работе 2, вариант 1

## Тема

Регистрация показаний аналогового датчика температуры с последующим сглаживанием результатов.

## Цель работы

Изучить методы считывания аналоговых сигналов с датчика температуры `LM35`, организовать сбор выборки фиксированного объема по внешнему запуску и выполнить сглаживание результатов методом модифицированного скользящего среднего.

## Оборудование

- Arduino Uno
- датчик температуры `LM35`
- кнопка запуска
- макетная плата
- соединительные провода

## Схема подключения

- `LM35 VCC -> 5V`
- `LM35 VOUT -> A0`
- `LM35 GND -> GND`
- `Кнопка -> D2 и GND`

## Описание алгоритма

После подачи питания микроконтроллер находится в режиме ожидания внешнего сигнала запуска на входе `D2`. По прерыванию начинается процесс выборки. Сигнал датчика температуры считывается через `A0` с частотой `10 Гц`, а всего сохраняется `64` отсчета. Каждое значение преобразуется в температуру в градусах Цельсия. Для сглаживания используется модифицированное скользящее среднее, которое рассчитывается рекурсивно по предыдущему сглаженному значению.

## Основные формулы

### Преобразование к температуре

`U = ADC * 5 / 1023`

`T = U * 100`

Для `LM35` коэффициент преобразования составляет `10 мВ / °C`.

### Модифицированное скользящее среднее

`Y(k) = Y(k-1) + (X(k) - Y(k-1)) / N`

где:

- `X(k)` - текущее измерение;
- `Y(k)` - сглаженное значение;
- `N` - параметр окна, в работе принят равным `5`.

## Результат работы программы

Программа формирует три массива:

- исходные значения АЦП;
- значения температуры без сглаживания;
- сглаженные значения температуры.

После завершения сбора они остаются в памяти микроконтроллера и могут быть считаны средствами отладки. При поступлении нового внешнего сигнала программа выполняет повторную выборку с обновлением содержимого массивов.

## Вывод

В ходе лабораторной работы была реализована система регистрации температуры с внешним запуском. Полученные данные сохраняются в памяти микроконтроллера, а применение модифицированного скользящего среднего уменьшает влияние случайных колебаний измеряемого сигнала.
