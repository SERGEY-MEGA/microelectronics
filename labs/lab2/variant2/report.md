## Тема

Дискретный сбор и статистическая обработка показаний аналогового датчика с выдачей данных в монитор порта.

## Цель работы

Освоить считывание аналоговых сигналов с потенциометра, реализовать программный антидребезг кнопки, организовать хранение последних измерений в кольцевом буфере и вычислять основные статистические характеристики на скользящем интервале.

## Оборудование

- Arduino Uno
- потенциометр
- кнопка
- макетная плата
- соединительные провода

## Схема подключения

- `Потенциометр: 5V - A0 - GND`
- `Кнопка: D2 - GND`

## Описание алгоритма

По нажатию кнопки контроллер считывает значение с аналогового входа `A0`. Перед фиксацией нажатия выполняется программный антидребезг с временной задержкой `50 мс`. Каждое новое измерение сохраняется в кольцевом буфере размером `10` элементов. После обновления буфера программа пересчитывает среднее арифметическое и среднеквадратичное отклонение по всем сохраненным значениям. По команде пользователя через `Serial Monitor` результаты выводятся в текстовом виде.

## Основные формулы

### Среднее арифметическое

`M = (x1 + x2 + ... + xn) / n`

### Среднеквадратичное отклонение

`sigma = sqrt(((x1 - M)^2 + ... + (xn - M)^2) / n)`

## Результат работы программы

Система сохраняет не более десяти последних измерений. При переполнении буфера самое старое значение автоматически заменяется новым. Пользователь может в любой момент открыть монитор порта и запросить текущие данные командой `d`.

## Вывод

В ходе работы был реализован дискретный сбор аналоговых данных с подавлением ложных срабатываний кнопки. Кольцевой буфер позволил организовать скользящее окно измерений, а расчет среднего и стандартного отклонения обеспечил базовую статистическую обработку сигнала.
