# ЛР2, вариант 2

## Задача

Организовать дискретный сбор данных с потенциометра по нажатию кнопки с программным антидребезгом. Для десяти последних измерений вычислять среднее арифметическое и среднеквадратичное отклонение. Выводить данные в монитор порта по запросу пользователя.

## Используемые компоненты

- Arduino Uno
- потенциометр
- кнопка
- макетная плата
- соединительные провода

## Подключение потенциометра

- один крайний вывод -> `5V`
- второй крайний вывод -> `GND`
- средний вывод -> `A0`

## Подключение кнопки

- один контакт кнопки -> `D2`
- второй контакт кнопки -> `GND`

Используется `INPUT_PULLUP`, поэтому внешний подтягивающий резистор не требуется.

## Таблица соединений

| Элемент | Контакт элемента | Arduino Uno |
|---|---|---|
| Потенциометр | крайний вывод 1 | 5V |
| Потенциометр | средний вывод | A0 |
| Потенциометр | крайний вывод 2 | GND |
| Кнопка | 1 | D2 |
| Кнопка | 2 | GND |

## Алгоритм работы

1. По нажатию кнопки выполняется одно измерение напряжения на потенциометре.
2. Кнопка обрабатывается с программным антидребезгом.
3. Новое значение заносится в кольцевой буфер на `10` элементов.
4. После каждого измерения пересчитываются:
   - среднее арифметическое;
   - среднеквадратичное отклонение.
5. При вводе команды `d` в Serial Monitor выводятся накопленные измерения и статистика.

## Команды монитора порта

- `d` - вывести значения буфера и статистику
- `h` - вывести справку

Скорость монитора порта: `9600 бод`.

## Как проверять

1. Соберите схему.
2. Откройте папку `lab2_variant2` в Arduino IDE и загрузите скетч.
3. Откройте Serial Monitor.
4. Нажимайте кнопку, изменяя положение потенциометра.
5. Введите `d` и проверьте вывод последних измерений, среднего и стандартного отклонения.
