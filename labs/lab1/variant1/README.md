# ЛР1, вариант 1

## Задача

Реализовать вывод 6-битного числа в двоичном коде через `Port B` на линейке из 6 светодиодов. Инкрементирование и декрементирование выполнять двумя кнопками.

## Используемые компоненты

- Arduino Uno
- 6 светодиодов
- 6 резисторов `220 Ом`
- 2 кнопки
- макетная плата
- соединительные провода

## Пины `Port B` на Arduino Uno

У Arduino Uno младшие шесть разрядов порта `B` выведены на цифровые пины:

- `PB0 -> D8`
- `PB1 -> D9`
- `PB2 -> D10`
- `PB3 -> D11`
- `PB4 -> D12`
- `PB5 -> D13`

## Подключение светодиодов

Для каждого светодиода:

- анод -> цифровой вывод `D8-D13` через резистор `220 Ом`
- катод -> `GND`

## Подключение кнопок

- кнопка инкремента -> `D2` и `GND`
- кнопка декремента -> `D3` и `GND`

Используется `INPUT_PULLUP`.

## Таблица соединений

| Светодиод | Arduino Uno |
|---|---|
| LED0 | D8 |
| LED1 | D9 |
| LED2 | D10 |
| LED3 | D11 |
| LED4 | D12 |
| LED5 | D13 |

| Кнопка | Arduino Uno |
|---|---|
| INC | D2 и GND |
| DEC | D3 и GND |

## Логика работы

1. На светодиодах отображается текущее 6-битное число.
2. Нажатие кнопки `INC` увеличивает число на единицу.
3. Нажатие кнопки `DEC` уменьшает число на единицу.
4. Для диапазона `0..63` используется циклический переход.

## Как проверять

1. Соберите схему.
2. Откройте папку `lab1_variant1` в Arduino IDE и загрузите скетч.
3. Нажимайте кнопки `INC` и `DEC`.
4. Убедитесь, что на линейке отображается двоичный код числа.
