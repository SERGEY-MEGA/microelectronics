# Отчет по лабораторной работе 1, вариант 2

## Тема

Формирование динамической светодиодной индикации на основе дискретных сигналов.

## Цель работы

Изучить методы управления светодиодной линейкой и реализовать эффект бегущего огня с изменяемой скоростью по нажатию кнопок.

## Оборудование

- Arduino Uno
- 6 светодиодов
- 6 резисторов `220 Ом`
- 2 кнопки
- макетная плата
- соединительные провода

## Схема подключения

- `D8-D13 -> 220 Ом -> аноды светодиодов`
- `катоды светодиодов -> GND`
- `FAST: D2 -> кнопка -> GND`
- `SLOW: D3 -> кнопка -> GND`

## Описание алгоритма

Светодиоды подключены к младшим линиям `Port B`. В каждый момент времени включен только один светодиод. Через заданный интервал времени активный светодиод последовательно перемещается по линейке, после чего цикл повторяется. Кнопки изменяют длительность паузы между шагами, тем самым регулируя скорость эффекта. Для исключения ложных срабатываний применяется программный антидребезг.

## Результат работы программы

После запуска программы на светодиодной линейке формируется бегущий огонь. Пользователь может изменять скорость движения в широких пределах с помощью двух кнопок.

## Вывод

В работе была реализована динамическая индикация на дискретных выходах микроконтроллера и освоены приемы программного управления скоростью световых эффектов.
