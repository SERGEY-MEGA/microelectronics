# ЛР1, вариант 2

## Задача

Реализовать бегущий огонь на линейке из 6 светодиодов. Скорость движения должна увеличиваться и уменьшаться по нажатию кнопок.

## Используемые компоненты

- Arduino Uno
- 6 светодиодов
- 6 резисторов `220 Ом`
- 2 кнопки
- макетная плата
- соединительные провода

## Подключение светодиодов

Для удобства используется та же группа выводов `D8-D13`, соответствующая младшим битам `Port B`.

- `D8 -> LED0`
- `D9 -> LED1`
- `D10 -> LED2`
- `D11 -> LED3`
- `D12 -> LED4`
- `D13 -> LED5`

Каждый анод подключается через резистор `220 Ом`, катоды соединяются с `GND`.

## Подключение кнопок

- кнопка уменьшения скорости -> `D2` и `GND`
- кнопка увеличения скорости -> `D3` и `GND`

Используется `INPUT_PULLUP`.

## Алгоритм работы

1. На линейке горит один светодиод.
2. Через заданный интервал активный светодиод сдвигается к следующему.
3. Кнопка на `D2` увеличивает задержку между шагами, то есть замедляет бегущий огонь.
4. Кнопка на `D3` уменьшает задержку между шагами, то есть ускоряет бегущий огонь.

## Параметры

- начальная задержка: `250 мс`
- минимальная задержка: `50 мс`
- максимальная задержка: `1000 мс`
- шаг изменения задержки: `50 мс`

## Как проверять

1. Соберите схему.
2. Откройте папку `lab1_variant2` в Arduino IDE и загрузите скетч.
3. Наблюдайте бегущий огонь.
4. Кнопками изменяйте скорость его движения.
