# Отчет по лабораторной работе 4, вариант 2

## Тема

Построение системы управления с обратной связью для индикации уровня освещенности.

## Цель работы

Изучить работу фоторезистора в составе делителя напряжения, освоить считывание аналогового сигнала и организовать управление светодиодной шкалой в зависимости от измеренного уровня освещенности.

## Оборудование

- Arduino Uno
- фоторезистор `LDR GL5516`
- резистор `10 кОм`
- 8 светодиодов
- 8 резисторов `220 Ом`
- макетная плата
- соединительные провода

## Схема подключения

- `+5V -> LDR -> A0 -> 10 кОм -> GND`
- `D6-D13 -> 220 Ом -> аноды светодиодов`
- `катоды светодиодов -> GND`

## Описание алгоритма

Фоторезистор и резистор `10 кОм` образуют делитель напряжения. Микроконтроллер считывает напряжение в средней точке делителя через вход `A0`. Для подавления резких скачков применяется простое рекурсивное сглаживание. После этого значение масштабируется в диапазон от `0` до `8`, соответствующий числу светодиодов на панели индикации. Чем выше освещенность, тем больше светодиодов включается.

## Результат работы программы

Система работает как простая светодиодная шкала. При увеличении освещенности на фоторезисторе растет аналоговое значение, и на индикаторе загорается больше светодиодов. При уменьшении освещенности количество светящихся светодиодов уменьшается.

## Вывод

В ходе лабораторной работы была реализована система управления с обратной связью по уровню освещенности. Использование аналогового датчика и светодиодной шкалы позволило наглядно показать зависимость выходного воздействия от входного измеряемого сигнала.
