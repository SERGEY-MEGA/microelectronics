# ЛР4, вариант 2

## Задача

Реализовать индикатор уровня освещенности, в котором число включенных светодиодов зависит от сигнала фоторезистора `LDR GL5516`.

## Используемые компоненты

- Arduino Uno
- фоторезистор `LDR GL5516`
- резистор `10 кОм`
- 8 светодиодов
- 8 резисторов `220 Ом`
- макетная плата
- соединительные провода

## Схема делителя напряжения

- `+5V -> LDR -> A0 -> 10 кОм -> GND`

При увеличении освещенности сопротивление фоторезистора уменьшается, напряжение на `A0` возрастает, и количество светящихся светодиодов увеличивается.

## Подключение светодиодов

Для каждого светодиода:

- анод -> цифровой вывод Arduino через резистор `220 Ом`
- катод -> `GND`

## Таблица соединений

| Светодиод | Arduino Uno |
|---|---|
| LED1 | D6 |
| LED2 | D7 |
| LED3 | D8 |
| LED4 | D9 |
| LED5 | D10 |
| LED6 | D11 |
| LED7 | D12 |
| LED8 | D13 |

## Подключение фоторезистора

| Элемент | Контакт элемента | Arduino Uno |
|---|---|---|
| LDR | вывод 1 | 5V |
| LDR | вывод 2 | A0 |
| Резистор 10 кОм | вывод 1 | A0 |
| Резистор 10 кОм | вывод 2 | GND |

## Алгоритм работы

1. Программа постоянно считывает аналоговое значение с входа `A0`.
2. Для уменьшения рывков используется простое цифровое сглаживание.
3. Значение освещенности преобразуется в число от `0` до `8`.
4. На светодиодной шкале зажигается соответствующее количество светодиодов.

## Как проверять

1. Соберите схему.
2. Откройте папку `lab4_variant2` в Arduino IDE и загрузите скетч.
3. Освещайте и затемняйте фоторезистор.
4. Убедитесь, что число включенных светодиодов изменяется пропорционально освещенности.
